Die API-Dokumentation des Adventure PHP Frameworks ist sowohl als Online- als auch als Dokumentation zum Download verfügbar.

Die API-Dokumentation des Adventure PHP Frameworks ist sowohl als Online- als auch als Dokumentation zum Download verfügbar. Die HTML-Dokumentationen wurden mit Hilfe von doxygen (http://doxygen.org) und dem Grafik-Modul graphviz (http://www.graphviz.org/) erstellt.

Um eine optimale Dokumentation zu garantieren werden die API-Dokumentationen Release-abhängig generiert. Im Anschluss finden Sie eine Liste der verfügbaren Online-API-Dokumentationen. Die kompletten Dokumentationen zum Download finden Sie unter beim gewünschten Release.
