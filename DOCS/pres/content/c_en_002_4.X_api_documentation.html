The api documentation of the Adventure PHP Framework is available in two different flavours: online and offline documentation. The docs are generated by doxygen (http://doxygen.org) featuring graphviz (http://www.graphviz.org/).

To provide best compatibility with each release the documentation is generated for each release separately. Please have a look at the following list to find the documentation suitable for your release. For downloads, please refer to for the desired release.
