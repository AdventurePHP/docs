Eine sehr häufige Anwendung auf einer Webseite ist ein Kontaktformular. Das Modul "Kontaktformular" bietet hier eine fertige und konfigurierbare Lösung an, die nur noch verwendet werden muss. Das Formular kann auf einer dafür vorgesehenen Seite mit <core:importdesign namespace="modules::kontakt4::pres::templates" template="contact" />

eingebunden werden. Eine ausführliche Beschreibung zur Benutztung und zur Implementierung findet sich im Bereich unter .

Bitte beachten Sie die Hinweise zur HTML-Formatierung des Moduls unter !

Das Kontakt-Formular unterstützt seit Release 1.15 per Konfiguration Sprach-abhängige Templates für den Versand der Bestätigungs- und Benachrichtigungs-E-Mails. Die zu verwendenden Templates werden in der Konfigurationsdatei

/config/modules/kontakt4/{CONTEXT}/{ENVIRONMENT}_mail_templates.ini

erwartet. Die Templates werden für jede Sprache in einer separaten Sektion definiert:

[de] confirmation.namespace = "modules::kontakt4::pres::templates::mail" confirmation.template = "confirmation_de" notification.namespace = "modules::kontakt4::pres::templates::mail" notification.template = "notification_de" [en] confirmation.namespace = "modules::kontakt4::pres::templates::mail" confirmation.template = "confirmation_en" notification.namespace = "modules::kontakt4::pres::templates::mail" notification.template = "notification_en"

Als Vorlage kann die Beispiel-Konfiguration EXAMPLE_mail_templates.ini unter /config/modules/kontakt4/ aus dem apf-configpack-*-Release-Dateien genutzt werden.
