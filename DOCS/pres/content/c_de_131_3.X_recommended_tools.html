PHPStorm IDE
Als Entwicklungs-Umgebung empfehlen wir PHPStorm IDE. Sie verfügt über eine ausgefeilte Projekt-Verwaltung, ausgezeichnete Unterstützung für das Schreiben und Refactoren von Code sowie eine exzellente Integration von xDebug.
Visual Paradigm SDE (UML Modellierung)
Das UML-Tool SDE UML von Visual Paradigm ergänzt die Netbeans IDE um ein ausgereiftes Modellierungstool. Die frei erhältliche Community Edition ist ein ideales Tool zur Unterstützung der objektorientierten Entwicklung.
xDebug
Die PHP-Extension xDebug ermöglicht es, die Ausführung von PHP-Code im single step Modus zu debuggen. Zusammen mit der Netbeans IDE können während der Programmausführung beliebige Stellen mit Haltepunkten versehen und der aktuelle Zustand der Applikation eingesehen werden.
KCacheGrind und WinCacheGrind
Die beiden Tools KCacheGrind und WinCacheGrind sind nützliche Helfer, wenn es um die Performance-Optimierung von PHP-Anwendungen geht. Sie analysieren cachegrind Trace-Dateien, die von xDebug erzeugt werden und liefern aussagekräftigere Hinweise auf Flaschenhälse einer Anwendung.
