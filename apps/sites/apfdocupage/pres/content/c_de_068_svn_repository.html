Subversion repository

Seit dem Release 1.7-beta steht ein Subversion-Repository zur Verfügung, über das der aktuelle Entwicklungsstand bezogen werden kann. Code-Dateien aus dem Repository sind grundsätzlich als unstable zu betrachten!.

Unter http://adventurephpfra.svn.sourceforge.net/viewvc/adventurephpfra/trunk/ steht ein HTTPs-Zugang zum Subversion-Repository bereit, über 
  svn co https://adventurephpfra.svn.sourceforge.net/svnroot/adventurephpfra adventurephpfra
kann eine Kopie des Repositories bezogen werden.
