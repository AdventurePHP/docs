The comment function modul is a modul that can be configured and included in each web page by adding an XML tag to an existing template. The comment function modul is a modul that can be configured and included in each web page by adding an XML tag: <core:importdesign namespace="modules::comments::pres::templates" template="comment" categorykey="{CAT_KEY}" /> The value "{CAT_KEY}" must be changed to something useful. A detailed description on the function of this modul can be found in the tutorials section under comment function.
