The sandbox is a tool to be able to start development immediately. The sandbox is an example web page which can be run without a database or external resources. Hence, it offers an ideal basis for the development of new modules or applications. The online sandbox page is available under sandbox.adventure-php-framework.org.



The source code files of the sandbox can be downloaded on the downloads page. In order to install the sandbox files locally simply select the suitable adventure-demopack-* file and unpack it under DocumentRoot of your web server. For PHP4 and PHP5 each release different packages are available. Other tips can be taken from the README.
