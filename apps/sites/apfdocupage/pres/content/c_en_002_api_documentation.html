The api documentation of the adventure PHP framework is available in two different flavours: online and offline documentation. The docs are generated by doxygen featuring graphviz and there are pure html or chm documentation files. The links to the online api documentation and the downloadable versions can be found on the download page. The api documentation of the Adventure-PHP-Framework is available in two different flavours: online and offline documentation. The docs are generated by doxygen (http://doxygen.org) featuring graphviz (http://www.graphviz.org/) and there are pure html or chm documentation files.
Due to changes in release management, the api documentation was moved to the download page and bundled with every release available. This procedure guaranteed, that each version of the framework is well documented. Please change to the download page to view the documentation.
