Dieses HOWTO beschreibt die Installation des APF über RPM-Pakete die von Reiner Rottmann zur Verfügung gestellt werden. Sie erleichtern den Versions-Upgrade und die Verteilung auf RedHat basierten LINUX-Distributionen und solchen, die YUM unterstützen. Reiner Rottmann, Spezialist für LINUX Cluster Systeme bei der ATIX AG, hat zur einfacheren Installation des APF auf RedHat-basierten LINUX-Distributionen einen Satz an RPMs (RedHat Package-Format) bereitgestellt. Diese werden für jedes Release ab 1.10 über einen eigenen YUM-Channel bereitgestellt. Die Installation und die Verteilung der einzelnen Pakete kann deshalb sehr einfach über die in den RedHat-Distributionen enthaltenen Mechanismen vorgenommen werden.


1. Konfiguration

Zur Konfiguration muss ein neues YUM-Repository angelegt werden. Dies kann entweder per [root@centos53 ~]# cd /etc/yum.repos.d/ [root@centos53 yum.repos.d]# wget http://www.rottmann.it/apf/apf.repo oder durch manuelles Anlegen einer *.repo-Datei im Verzeichnis /etc/yum.repos.d. Diese muss den folgenden Inhalt haben: [root@centos53 yum.repos.d]# cat /etc/yum.repos.d/apf.repo # Name: Adventure PHP Framework Repository for Red Hat Enterprise 5 # URL: http://www.adventure-php-framework.org # Maintainer: Reiner Rottmann ( reiner [at] rottmann . it ) # [apf] name = Adventure PHP Framework baseurl = http://www.rottmann.it/apf/ enabled = 1 protect = 1 gpgkey = http://www.rottmann.it/apf/RPM-GPG-KEY.key gpgcheck = 0

2. Installation
