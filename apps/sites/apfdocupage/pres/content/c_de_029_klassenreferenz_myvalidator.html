Klassen-Referenz - myValidator

Die Klasse myValidator bildet die Grundlage für die Validierung der Formular-Controls des Frameworks. Sie bietet Methoden zur Validierung von diversen Benutzereingaben wie Text, E-Mail und Telefon- sowie Fax-Nummern. Falls eine Applikation Validierungsaufgaben ausserhalb der Formular-TagLibs durchführen muss, kann die Klasse nach der Einbindung via import('tools::validator','myValidator'); mit folgenden statischen Methoden aufwarten:
  • validateText():
    Validiert, ob der angegebene Text mindestend 3 Zeichen enthält.
  • validateEMail():
    Prüft, ob der gegebene String eine syntaktisch richtige E-Mail repräsentiert.
  • validateTelefon():
    Validiert, ob der gegebene String eine syntaktisch richtige Telefon-Nummer darstellt.
  • validateFax:
    Prüft, ob der gegebene String eine syntaktisch richtige Fax-Nummer darstellt.
  • validateNumber():
    Validiert, ob der gegebene String eine Nummer ist.
  • validateFolder():
    Prüft, ob der gegebene Wert für einen Ordnernamen geeignet ist.
  • validateRegExp():
    Validiert einen Wert gegen einen regulären Ausdruck.
