This page lists interesting literature pages that can be used to work with apf. Further, online ressources are depicted.

1. Common literature

In most cases, the documentation, information or examples on this page is detailed enought to build your own website or web application. To get a little deeper into the software architecture and software design issue it is recommended to read a book or magazine dealing with design pattern. For this reason, the following list can be used as such a ressource:
  • Wikipedia
  • Uni Rostock (german)
  • Papier „Entwurf und Implementierung eines Webportals zur Registrierung und Verwaltung von Weiterbildungsveranstaltungen auf Basis von MySQL und PHP“ (german)
Beyond this online ressources the author advices the following books:
  • Patterns of enterprise application architecture - Martin Fowler (Reference guide)
  • Refactoring - Martin Fowler (Software refactoring und analysis)
  • UML distilled - Martin Fowler (UML guide with many real life examples)
  • Softwaretechnik Praxiswissen - Johannes Siedersleben (Reference guide on software development processes)

2. PHP introduction

To get a basic understanding of programming with PHP you should have a look at:
  • PHP-FAQ
  • Quake-Net-PHP-FAQ
  • PHP-Coding-Standard.de (Recommondations on PHP coding conventions)
