Eine sehr häufige Anwendung auf einer Webseite ist ein Kontaktformular. Das Modul "Kontaktformular" bietet hier eine fertige und konfigurierbare Lösung an, die nur noch verwendet werden muss. Das Formular kann auf einer dafür vorgesehenen Seite mit 
  <core:importdesign
    namespace="modules::kontakt4::pres::templates"
    template="kontakt"
   />
eingebunden werden. Eine ausführliche Beschreibung zur Benutztung und zur Implementierung findet sich im Bereich Tutorials unter Kontakt-Formular.
