Seit dem Release 1.7-beta steht ein Subversion-Repository zur Verfügung, über das der aktuelle Entwicklungsstand bezogen werden kann. Since release 1.7-beta a subversion repository is available. There cou can obtain a copy of the current development state. Please be aware, that the code in the trunk area of the repository is to be considered unstable!.

Under http://adventurephpfra.svn.sourceforge.net/viewvc/adventurephpfra/trunk/ you can browse the subversion repository via HTTPs, with the command svn co https://adventurephpfra.svn.sourceforge.net/svnroot/adventurephpfra adventurephpfra you can checkout the current development revision.
