Subversion repository

Since release 1.7-beta a subversion repository is available. There cou can obtain a copy of the current development state. Please be aware, that the code in the repository is to be considered as unstable!.

Under http://adventurephpfra.svn.sourceforge.net/viewvc/adventurephpfra/trunk/ you can browse the subversion repository via HTTPs, with the command 
  svn co https://adventurephpfra.svn.sourceforge.net/svnroot/adventurephpfra adventurephpfra
you can checkout the current development revision.
