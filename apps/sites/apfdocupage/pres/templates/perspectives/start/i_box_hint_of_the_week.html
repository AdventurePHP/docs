Mit dem APF-Relase 1.7-RC2 wird das Sichern von Formularen vor SPAM einfach. Das Modul captcha bietet eine Formular-TagLib, die einfach per XML-Tag zum Formular hinzugefügt werden kann.
Mehr über das Modul, die Konfiguration und Verwendung kann auf der Seite Mitgelieferte Module nachgelesen werden.


Since release 1.7-RC2, securing web forms with the APF becomes easy. The captcha module therefore includes a form taglib, that can be included using the appropriate XML tags.
More information abaout the module, the configuration and usage can be obtained on the builtin modules page.
