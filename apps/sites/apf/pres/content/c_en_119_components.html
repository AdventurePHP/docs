The pages listed below are intended to give you a brief introduction into development with the APF. In case these are not enough have a look at the page.

The framework's core

The core of the framework merely consists of the page controller implementation and some more tools, that are responsible for the integration of the components.

Extensions

Since the end of 2009, enganged APG user have provided their extensions to the community. These extensions are directly packaged with the APF release und are thus maintained with with each release. The documentation of the extensions can be found within the extensions folder of the release package and the Wiki.
