Da Anwendungen im Adventure PHP Framework per XML-Tag in eine bestehende Seite eingebunden werden, kann das Gästebuch auf der gewünschten Seite mit einem <core:importdesign namespace="modules::guestbook::pres::templates" template="guestbook" guestbookid="{GB_ID}" />

aktiviert werden. Der Wert {GB_ID} muss dabei auf die ID des anzuzeigenden Gästebuchs gesetzt werden. Eine ausführliche Beschreibung zur Benutztung und zur Implementierung findet sich im Bereich Tutorials unter .
