The stringEncryptor provides functions to create sting hashes. Merely this component can be used as a device to "encrypt" password stings. After adding a import('tools::string','stringEncryptor'); to the program's code, tha component can be created by using the following code part: $sE = &$this->__getServiceObject('tools::string','stringEncryptor'); echo $sE->getPasswordHash('=MEiN&Pa$$W0rt='); To create the hash correctly the class needs the configuration file {ENVIRONMENT}_encryption.ini to be stored under the namespace /config/tools/string/{CONTEXT} This file defines the password salt for the creation of the hash value by use of the PHP function crypt(). In common the config file contains the following lines: [Standard] PasswortSalt = ""
Note: Further information concerning the definition of password salts can be found on http://de.php.net/manual/de/function.crypt.php.
