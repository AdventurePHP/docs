The dateTimeManager is a resource to create and manupulate date and time data.

The dateTimeManager is deprecated starting with Release 1.15 and will be removed in 1.16. Please use the PHP class DateTime in order to create, manipulate and display date and time values.

To ease migration, all methods will raise deprecated errors.

The dateTimeManager is a resource to create and manipulate date and time data. After including the class via

import('tools::datetime','dateTimeManager'); the followinf static methods are avaliable:
  • calculateTimeDifference():
    Calculates a target time based on a start time and a difference definition.
  • calculateDate():
    Calculates a target date based on a start date and a difference definition.
  • getMonthInfo():
    Returns information about a given month.
  • generateGermanDate():
    Generates a date of the form 00.00.0000.
  • generateDate():
    Generates a date of the form 0000-00-00.
  • generateTime():
    Generates a time of the form 00:00:00.
  • generateTimeStamp():
    Generates a time stamp of the form 0000-00-00 00:00:00'.
  • addLeadingZero():
    Adds a leading zero to a time value if necessary.
  • showMonthLabel():
    Zeigt den Anzeigenamen eines Monats an. Die Methode beherrscht aktuell nur die Sprache Deutsch.
  • convertDate2Normal():
    Convertes a date data of the form 0000-00-00 into the form 00.00.0000.
  • splitDate():
    Splits a date of the form 0000-00-00 or of the form 00.00.0000 into it's parts and returns an array containing this values.
  • splitTime():
    Splits a time of the form 00:00:00 into it's parts and returns an array containing this values.
