The sandbox is a tool to be able to start development immediately.

The sandbox is an example web page which can be run without a database or external resources. Hence, it offers an ideal basis for the development of new modules or applications.

The source code files of the sandbox can be downloaded on the page. In order to install the sandbox files locally simply select the suitable apf-demopack-* file and unpack it under DocumentRoot of your web server. Other tips can be taken from the README.
