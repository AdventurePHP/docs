Die Sandbox gibt dem Entwickler ein Tool an die Hand mit der die Entwicklung erleichtert wird.

Die Sandbox ist eine Beispiel-Webseite, die ohne Datenbank und externen Ressourcen betrieben werden kann. Sie bietet daher eine ideale Grundlage für die Entwicklung neuer Module oder Applikationen.

Die Quellcode-Dateien der Sandbox können unter Downloads heruntergeladen werden. Zur lokalen Installation einfach die passende adventure-demopack-*-Datei auswählen und diese unterhalb des DocumentRoot des Webservers entpacken. Für PHP4 und PHP5 sind jeweils unterschiedliche Pakete verfügbar. Weitere Hinweise können dem README entnommen werden.
