Since release 1.7-beta a public subversion repository is available. There cou can obtain a copy of the current development state. Please be aware, that the code in the trunk area of the repository is to be considered unstable!.

Under http://sourceforge.net/p/adventurephpfra/code you can browse the subversion repository via HTTPS, with the command svn export http://svn.code.sf.net/p/adventurephpfra/code adventurephpfra-code you can export the current development revision.
