The adventure php framework understands itself as a utility to implement object oriented and generic PHP web applications. It supports the developer to create programms in compliance with approved software design pattern and the code base already has answers to many everyday problems. Further, it introduced many concepts and enfficiency tools already well known in the JAVA community.

Welcome!
