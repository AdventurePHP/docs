Since release 1.7-beta a public subversion repository is available. There cou can obtain a copy of the current development state. Please be aware, that the code in the trunk area of the repository is to be considered unstable!.

Under http://adventurephpfra.svn.sourceforge.net/viewvc/adventurephpfra/trunk/ you can browse the subversion repository via HTTPS, with the command svn export https://adventurephpfra.svn.sourceforge.net/svnroot/adventurephpfra adventurephpfra you can export the current development revision.

At present, the following snapshots are available:
  • 1.12 (PHP5)
  • 1.11 (PHP4)

Please note the structure of the APF repository described in the Wiki under http://wiki.adventure-php-framework.org/de/Struktur_des_APF-SVN_Repository.
