Die Klasse StringAssistant stellt statische Methoden zur Zeichenketten-Bearbeitung zur Verfügung. Hierunter fallen die Behandlung von Sonderzeichen in Dateinamen, kodierung von HTML-Zeichen und die Generierung von CAPTCHA-Zeichenfolgen.

Die Klasse StringAssistant stellt statische Methoden zur Zeichenketten-Bearbeitung zur Verfügung. Diese sind:

Methode Beschreibung
escapeSpecialCharacters() Entfernt Sonderzeichen, die für die Speicherung in der Datenbank und das spätere Anzeigen in HTML-Code Probleme machen.
replaceSpecialCharacters() Ersetzt die im Web für Dateinamen nicht zulässigen Sonderzeichen und gibt einen bereinigten String zurück.
encodeCharactersToHTML() Codiert eine Zeichenkette in HTML-Entities.
generateCaptchaString() Generiert einen String, der als Captcha-Prüfwort verwendet werden kann.

Die Anwendung der Methoden gestaltet sich wie folgt:

use APF\tools\string\StringAssistant; $sanitizedString = StringAssistant::escapeSpecialCharacters($_REQUEST['some-parameter']); $captchaChallenge = StringAssistant::generateCaptchaString(5); $entityEncoded = StringAssistant::encodeCharactersToHTML('foo@bar.com');
