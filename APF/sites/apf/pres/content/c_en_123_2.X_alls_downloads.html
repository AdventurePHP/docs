On this page, all releases of the Adventure PHP Framework are available. Besides, a detailed change log with hints to the relevant documentation pages, the API documentation and a link to the current SVN tag is presented.

Beneath the current version of the Adventure PHP Framework (APF), on this page you can download older releases.

Warning! Please note, that using older releases on production environments is not recommendend. Older versions may include unpatched security leaks. Therefor, please use older versions only if you know what you are doing or in case you have to (e.g. due to compatibility issues with the current version). Please consult the Components documentation page anyway!
