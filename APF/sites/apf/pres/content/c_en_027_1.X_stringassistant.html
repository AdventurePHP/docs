The class StringAssistant provides static methods to manipulate stings. Among these are special character removing from file names, HTML character encoding and the generation of CAPTCHA strings. The class StringAssistant provides static methods to manipulate stings. After importing the class via import('tools::string','StringAssistant'); the following functions can be used:
  • escapeSpecialCharacters() (static):
    Strips special characters that can be problematical during saving data to a database or while displaying them in form controls.
  • replaceSpecialCharacters() (static):
    Replaces characters that are not allowed for files names and returns a sanitized string.
  • encodeCharactersToHTML() (static):
    Encodes a string into HTML entities.
  • generateCaptchaString() (static):
    Generates a string, that can be used as a captcha check string.
