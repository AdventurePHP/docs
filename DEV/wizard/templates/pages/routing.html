Welcome to the APF documentation page setup wizard

This is a wizard to setup the APF documentation page on your local machine.

To get your setup done, please go through the following steps as they appear. In case your setup has been completed already, you may also execute certain stept to your need (e.g. refresh search index).

  • Adapt index.php: since the index.php file is prepared for production use and does not fit your local environment, please execute this wizard to adapt it to your needs.
  • Adapt database configuration: the database configuration in SVN is not prepared for your local environment. Hence, please use the database configuration wizard change settings as desired.
  • Setup databases: once you've configured all databases it's time to setup all necessary tables. In case you haven't completed this task yet, please start the setup wizard.
  • Initialize search index: the APF documentation page relies on a database-based search index to generate URLs. In case you want to setup the page running on your local machine or you've added pages, you can re-create or update the index here.

In case of any questions, please refer to Wiki page Local setup of the APF docs page.
