Adapt database configuration

The database connection configuration needs to be adapted to your local setup. Installation of forum, wiki, and tracker is not required. You can leave fields blank. Please fill in the following form to complete this step:

Host, port, user, and password is to be specified for all connections. In case you intend to use different users, please manually adapt APF/config/core/database/sites/apf/DEFAULT_connections.ini.

Database Comments stores user comments displayed at the end of the page.

Database FulltextSearch stores the search index as well as the article list used for link generation.

Database Forum hosts a local PHPBB installation (optional).

Database Wiki hosts a local WikiMedia installation (optional).

Database Tracker hosts a local Mantis installation (optional).

Database Stat stores page statistics locally (optional).
