Setup database tables

Using this wizard you can setup the table layout for all databases configured within the database configuration wizard. You can either create tables for all databases (recommended for initial setup) or just one (recommended for partial setup or DDL changes).

Please choose which database to initialize. You may either choose all or a particular one. Database setup is repeatable and thus creates tables only when not existing.

Database FulltextSearch stores the search index as well as the article list used for link generation. Please note, that this wizard will not setup local installations of Forum (PHPBB), Wiki (MediaWiki), and Tracker (Mantis).
